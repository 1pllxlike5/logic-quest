# 数字逻辑刷题软件 — 完整规划方案

> **简称**：LogicQuest · **风格**：游戏化闯关风（风格 C）· **架构**：纯前端单页应用

---

## 一、项目概述

将《数字电路与逻辑期末复习题》docx 题库转化为一款**游戏化刷题 Web 应用**，支持离线使用、跨网络部署。总计约 90+ 道题目，覆盖 5 大题型。

### 核心目标

- 🎮 **好玩**：闯关、连击、成就、金币让刷题像游戏
- 🌐 **到处能用**：离线本机、GitHub Pages、任意静态托管
- 📱 **多端适配**：PC + 平板 + 手机都能刷
- ⚡ **零依赖运行**：一个 HTML 搞定，双击即用

---

## 二、技术架构

| 层 | 技术选型 | 说明 |
|---|---------|------|
| 界面 | HTML5 + CSS3 + 原生 JS | 零框架依赖，单文件部署 |
| 数据 | 题库内嵌为 JS 对象 | 约 90+ 题，JSON 结构化 |
| 持久化 | localStorage | 进度、金币、成就、错题本地存储 |
| 图标 | Emoji + CSS 绘制 | 无需图标库 |
| 图表 | Canvas 自绘 | 统计看板用，不再引入 Chart.js |
| 部署 | 静态文件 | GitHub Pages / 本地 / 任意静态服务 |

### 部署方案：GitHub Pages（主方案）

通过 GitHub 仓库托管，自动生成公网链接，任何网络环境下都能打开刷题。

#### 部署流程

```
本地开发 → git push → GitHub 仓库 → 开启 Pages → 获得公网链接
```

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 创建 GitHub 仓库 | 如 `logic-quest`，设为 **Public** |
| 2 | 推送代码 | 将 `index.html` push 到仓库 |
| 3 | 开启 Pages | Settings → Pages → Source: `main` 分支 → Save |
| 4 | 获取链接 | 等待 1-2 分钟，得到 `https://<用户名>.github.io/logic-quest/` |
| 5 | 分享使用 | 链接发给任何人，浏览器打开即用 |

#### 分享方式

| 方式 | 操作 | 适用场景 |
|------|------|---------|
| 🔗 直接发链接 | 微信/QQ 复制 `https://xxx.github.io/logic-quest/` | 最常用 |
| 📱 二维码 | 链接生成二维码，手机扫码打开 | 教室投屏、打印贴纸 |
| ⭐ GitHub Star | 同学 Star 仓库后在自家仓库找到 | 同班传播 |
| 📋 嵌入链接 | 放在课程网站/飞书文档/Notion 中 | 课程主页 |

#### 更新流程

```
本地修改 index.html → git commit → git push → GitHub Pages 自动更新（1-2分钟生效）
```

#### 补充：离线备用

| 场景 | 方案 |
|------|------|
| 无网络/GitHub 被墙 | 手机/电脑提前下载 `index.html`，本地双击打开 |
| U盘携带 | 拷贝 `index.html` 到 U 盘，任何电脑即插即用 |
| 机房断网 | 提前存一份到机房电脑桌面 |
